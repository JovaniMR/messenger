# CHAT REALTIME

#### Tecnologías
- HTML, CSS, Boostrap, Jvascript
- Vue js (2.x)
- Laravel (5.8)

#### Características
- Registro, inicio de sesión y edición de perfil
- Envia y recibe mensajes en tiempo real
- Implementación de emojis
- Etiqueta a los contactos en linea u offline
